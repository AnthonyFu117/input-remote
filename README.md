# Keyboard Remote
A keyboard sharing tool through different devices

## Implement

#### Server
*Python3.5* with
> [Tornado](http://www.tornadoweb.org/en/stable/)

#### Client/Windows
C# with WinForm
> [websocket-sharp](https://github.com/sta/websocket-sharp)

#### Communication
Websocket with Json
